Project files for ffDiaporama are XML files with .ffd extension.
  • This XML file does not contain copies of the multimedia elements (image, video, music) which are included in the project but only links to these files.
  • The links are relative to the location of the .ffd file. This ensures that if the directory containing the project file (.ffd ) also contains the multimedia elements (directly or in subdirectories), it is possible to move and/or to rename the directory.
  • Another advantage is that a project created under Linux can be opened under Windows and conversely.
