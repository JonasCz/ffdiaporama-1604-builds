This dialog box allows you to export the project and all ressources used by the project to a folder
 
Destination folder Allows you to select the base destination folder. Click on the folder button to open the directory selector.
Project subfolder Allows you to select the folder name to be created for the project files. By default the project title is proposed.
 
Once completed your selections, click the OK button to start the export.
At any time during this process, you can click the cancel button to stop the export.

Remarks:

  • The export takes into account all the resources that compose the project:
    • Images
    • Videos
    • Music
    • Thumbnail elements
    • Wallpapers
    • clipart that are not stored under the ClipArt alias.

  • Export does not just only copy the files. The following rules apply:
    • If multiple files from different directories have the same names, these files are re-named.
    • A .ffd project file is created taking into account any changes in directory and file name.
