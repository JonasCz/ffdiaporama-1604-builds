This dialog allows you to select a model and define chapter information for predefined title slides.
 
To understand the differences between Predefined title slides and Standard slides, see: Predefined title slide
 
At the top of the box, the following two settings allow you to filter the Predefined title slide that are displayed in the selection area.
Slide type You can select between:
Project title This type of slide is usually used early in the project as the opening credits.
The texts variables that are used by the models of this slide type are based only on project information. For more information on project information, see: Project properties dialog
Chapter title This type of slide is usually used several times in the project to mark the beginning of the chapter.
The texts variables that are used by the models of this slide type are based on project information and on chapter information.
Credit title This type of slide is usually used at the end of the project as the end credits.
The texts variables that are used by the models of this slide type are generally computed. For more information on texts variables, see: Dynamic variables
Category You can select between:
Statics models This type lists statics models ie without animations.
Animated models This type lists animated models ie with several shots.
Custom models This type lists the templates that you saved with the Save as model function of the Slide properties dialog box.
 
The selection area display models corresponding to the current Type and Category you previously selected.
  • The models are displayed and played, so you can see the slide as it will appear in your project with the content of dynamic variables.
  • The slide duration is displayed on the bottom of each thumbnails model.
 
When your selection of model is made, the following settings are available if slide type is Chapter model:
Chapter name Enter here the name of the chapter.
Chapter date different from project date If this box is checked, the date of the chapter is different from the date of the project.
Remark: It is often useful to differentiate the date of the chapter of the date of the project.
For example, if the project is My vacation from 1 to 15, the chapter may be limited to the 3.
Event date Select a date for the chapter.
Remark: This field is enabled only if the Chapter date different from project date check box is checked.
Override date Check this box to override the date and offer you to enter the date as a free multi-line text.
Remark: This field is enabled only if the Chapter date different from project date check box is checked.
Chapter location different from project location If this box is checked, the location of the chapter is different from the location of the project.
Remark: It is often useful to differentiate the location of the chapter of the location of the project.
For example, if the project location is where I rented for the holidays, the chapter location may be where I went on tour.
Event location The location where the chapter's event took place
  • Click on this button to select a location from the Add or modify a location dialog or using a favorite
  • Click on this button to remove actual location (set to empty)

Remark: These fields and buttons are enabled only if the Chapter location different from project location check box is checked.
 
Under these settings, the following two buttons allow you:
Project properties This button allows you to open Project properties dialog.
Convert to standard slide This button allows you to convert this slide to a standard slide which are fully customizable.
