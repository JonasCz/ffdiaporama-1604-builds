All slides have information chaptering so each slide can mark the beginning of a new chapter.

Remarks:

  • Except for the first chapter which start at 0, all other chapters begin after the entering transition.
  • The 1st slide of the mount always mark the beginning of a new chapter.
This dialog allows you to define settings for the chapter information of the slide:
 
Start a new chapter with this slide If this box is checked, the slide will mark the beginning of a new chapter.
Remark: This box cannot be deactivated for the first slide of the project.
Chapter name Enter here the name of the chapter.
Chapter date different from project date If this box is checked, the date of the chapter is different from the date of the project.
Remark: It is often useful to differentiate the date of the chapter of the date of the project.
For example, if the project is My vacation from 1 to 15, the chapter may be limited to the 3.
Event date Select a date for the chapter.
Remark: This field is enabled only if the Chapter date different from project date check box is checked.
Override date Check this box to override the date and offer you to enter the date as a free multi-line text.
Remark: This field is enabled only if the Chapter date different from project date check box is checked.
Chapter location different from project location If this box is checked, the location of the chapter is different from the location of the project.
Remark: It is often useful to differentiate the location of the chapter of the location of the project.
For example, if the project location is where I rented for the holidays, the chapter location may be where I went on tour.
Event location The location where the chapter's event took place
  • Click on this button to select a location from the Add or modify a location dialog or using a favorite
  • Click on this button to remove actual location (set to empty)

Remark: These fields and buttons are enabled only if the Chapter location different from project location check box is checked.
