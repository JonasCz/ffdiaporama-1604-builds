This dialog box allows you to define options for the application. It consists of the following 3 tabs:
  • The Application options tab allows you to define the general behavior of ffDiaporama.
  • The Project options tab allows you to define the options by default of the current project and for the new projects.
  • The Rendering options Tab allows you to define the options used by default during the generation of videos.

Application options

The Application options tab allows to define the general behavior of ffDiaporama.

Application options

Remember Windows size and positions If this box is checked, the sizes and the positions of all windows and dialogs will be saved and reused the next time you use ffDiaporama.
Remember last used folders If this box is checked, the directories for images, videos, projects, music and the generation of your movie will be saved and reused the next time you use ffDiaporama.
Enable "raster" Graphics system For Linux users only: If this box is checked, the graphic system "raster" is used instead of the default graphic system. This option is used only at application start up.
The "raster" Graphics accelerates the display but can raise problems on some computer in particular with KDE.
Disable tooltips in dialog box If this box is checked, tooltips will no longer appear in dialog boxes (only what's this will be available).
Offer to open the download Web page when a new version is available If this box is checked, a dialog box will appear at startup the next time a new version will be available.
By default, this box is checked for Windows users and unchecked for Linux users
Memory profil Defines the quantity of memory used as cache memory by ffDiaporama. This parameter is important because it allows to adapt ffDiaporama to your hardware.
For users of 32-bit systems, it is recommended that set this value to 512 MB if the computer has at least 2 GB of RAM.
For users of 64-bit systems, it is recommended that set this value to 2 GB if the computer has at least 4 GB of RAM.

Preview options

Frame rate Defines the number of images per second which will be generated to show the preview.
Note: the higher this number, the more smooth the preview will be. On the other hand, more computer resources will be required. Reduce this number on slow computers or if the gauge for your multimedia reader is frequently in the red zone.
Max video preview height Sets the maximum height of videos used for the preview. Beyond this height, the videos are automatically reduced for creating images.
This parameter affects only the preview and does not change the quality of rendered video.
It is recommended to reduce this value for slow computer.
Image smoothing (smooth images during preview) If this box is checked, the image smoothing effect will be applied to the images during the preview.
This effect consumes lots of resources and can result in slow preview on small machines. Unchecking this box allows then to keep a fluid preview.
This parameter affects only the preview and does not change the quality of rendered video.
Sound sample rate Defines the frequency of the sound generated to show the preview.
This parameter affects only the preview and does not change the quality of rendered video.
It is recommended to reduce this value for slow computer.

Editor options

Coordinates unit This option allows to select the unit for the coordinates which will be used in the properties dialog. You can choose:
  • Percentage: Coordinates (X, Y, Width and Height) are expressed in percentages
  • Pixel: Coordinates (X, Y, Width and Height) are expressed in pixels relative to a screen in 1080p mode
Object position This option allows you to adjust the positions occupied by new objects inserted into the timeline.
  • If Insert after the current selected object is selected, then the new slides will be created after the current slide.
  • If Append to the end of the project is selected, then the new slides will be created at the end of the project.
Object deletion If this box is checked, a confirmation prompt will be displayed every time you remove an object:
  • A slide from the main window
  • A shot or bloc from the slide properties dialog box
Video cropping If the Automatically crop 1920x1088 video to 1920x1080 is checked then video images are automatically cropped.
Remark: Some manufacturers produce FullHD videos 1920x1088 rather than 1920x1080. They make it to accelerate the encoding which is faster with a multiple of 16 lines. But this size is not 16:9. To keep the whole image, not only the image is deformed to fit into the screen but besides black bands appear on the sides. To avoid all these problems, by checking this box, these videos are truncated to be returned to the 1920x1080 sizes. Then 4 lines are cut at the top and bottom.
Video deinterlacing If the Activate deinterlace filter when add new MTS videos is checked, deinterlace filter will automatically be activated for all .mts videos added to the project.
Remark: Some manufacturers produce videos using interlaced mode. They make it to accelerate the encoding. The problem is it deforms the image on which moved lines appear. For more information on interlaced video, see this Wikipedia page: interlaced video

Network settings

Use a proxy to access the Internet Check this box if your network requires that you use a proxy.
Proxy address Enter address of the proxy host.
Proxy port Enter the port number to access the proxy or nothing to use default port.
Login Enter the login name to access the proxy or nothing if no login is required.
Password Enter the login password to access the proxy or nothing if no login is required.

Various options

Short date format This option allows you to specify the format in which computed dates are displayed.
Normally this format depends on the language used on your system, but it can be useful to change it especially if the interface is not available for your language.
Distance unit This option lets you specify the unit to be used for distances.

Project options

The Project options tab allows to define the options by default of the current project and for the new projects.

Project properties

Title Defined the automatic filling method to use for the title's project properties:
  • No automatic filling : You will have to define manually the title
  • Fill with project name when project save : The title will take the same name as the .ffd file
  • Fill with project folder name when project save : The title will take the same name as the folder in which is save the .ffd file
Album Defines the default value for the field Album
Author Defines the default value for the field Author that is often so called Artist by Media Center Systems
Default language Defines the default value for the field Language
Enhance id3v2 comptatibility For more compatibility with the standard id3v2, check this box to limit Title, Author and Album fields to 30 characters
Default thumbnail Defines the model use as default thumbnail. Thumbnails are images that are generated along with the videos. These images are used as a poster by most Media Center Systems.
For more information on joined files, see File attachments

Project options

New project geometry Selects the image geometry to be used for the creation of projects. Three geometries are available :
  • 4:3: Corresponds to an old TV with (almost) square screen
  • 16:9: Corresponds to a modern TV with rectangular screen
  • 2.35:1: Corresponds to the Cinema format (on a TV, this format will use black strips at the top and below).
Note: the geometry of a project is defined once and for all when it is created. It is not possible to modify it afterwards.
Default 1st shot duration Defines the default duration of the first shot for slides during their creation.
Next shot duration Defines the default duration of further shots.
Default transition Defines the transition to be inserted between inserted slides :
  • A transition chosen randomly from the library
  • No transition
  • Cross-fade transition

Default options for speed waves

Transition Sets the default speed wave for the transitions of new projects
Blocks animation Sets the default speed curve for the blocks animations of new projects
Images animation Sets the default speed for the image effects animations of new projects

Default options for new text block

This group of settings applies to the new text blocks created in slide dialogue by means of this button:
Text Allows to select the characters style applied to the new text blocks. This style defines the font, its size, its color and the shadow effects applied to the text.
Background Allows to select the style of background applied to the new text blocks. This style defines the transparency of the block and its filling.
Shape Allows to select the style of the shape to applied to the new text blocks. This style defines the shape (rectangle, circle, ...), the border, the opaqueness and the shadow effect applied to the block.
Coordinates Allows to select, for every project geometry, the style of the location applied to the new text blocks. This style defines the size, the coordinates, the rotations and the geometrical constraint.
Locking Allows to Select the geometric constraint applied to the new text blocks.
  • Unlock: The shape of the block is free
  • Lock to project geometry: The shape of the has the same shape as the image

Default options for new image or video block (when creating slide)

This group of settings applies to the new image or video blocks created from the main window by means of this button:
The following settings apply to all image formats:
Text style Allows to select the characters style applied to the text of the blocks. This style defines the font, its size, its color and the shadow effects applied to the text.
Shape Allows to select the style of the shape to applied to the new text blocks. This style defines the shape (rectangle, circle, ...), the border, the opaqueness and the shadow effect applied to the block.

For every image format and every project geometry select the style of location to applied to the new blocks. This style concerns the size, the coordinates, the rotations and the geometrical constraint.
Select, for each image format, the style of framing and the style of positioning to be applied to new blocks.
  • Framing styles define how to crop the image to make it compatible with the block in which is displayed.
  • The positioning styles define the the size, coordinates, rotations and geometric constraint to apply to blocks.

Default options for new image or video block (when adding block in slide dialog)

This group of settings applies to the new image or video blocks created in slide dialogue by means of this button:
The same selection that when creating slides apply.

Rendering options

The Rendering options Tab allows to define the options used by default during the generation of videos.

Default video name

This option allows to select the name which will be proposed by default during the rendering of videos. This option can be applied only if the project was saved at first.
Use project name as video name If the project name is "assembly.ffd", then "assembly" will be the proposed name
Use project folder's name as video name If the project name is "assembly.ffd" and is saved in the "My holidays 2011" directory, then "My holidays 2011" will be the proposed name
Use title project as video name Use the title defined in the project properties as the name of video

Default joined files options

For more information on joined files, see File attachments
Join thumbnail to rendered file If this box is checked then a thumbnail file will be created with video.
Join XBMC .nfo file to rendered file If this box is checked then a XMBC .nfo file will be created with video.

Default sound level

Set the volume level to be achieved by amplification for music and audio tracks when the volume is set to auto

Default device mode rendering options

Smartphone Selects the default type and equipment model when you will click the "Player and Smartphone" button of the "Render movie" menu
Multimedia system Selects the default type and equipment model when you will click the "Multimedia system" button of the "Render movie" menu
For the WEB Selects the default type and equipment model when you will click the "For the Web" button of the "Render movie" menu
Lossless Selects the default image size to be used when you will click the "Lossless" button of the "Render movie" menu
