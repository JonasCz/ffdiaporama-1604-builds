Predefined title slide are complete slides including formatting and animations.
The main interest of these slides is to standardize the slides within a project or across projects.

ffDiaporama included many Predefined title slides, but you can also define your own Predefined title slides.
To do this, create a standard slide using the Slide properties dialog box and save your slide as a model using:
  • Save as model function.

Remarks:

  • Predefined title slides are defined according to the following three categories:
    Project title This type of slide is usually used early in the project as the opening credits.
    The texts variables that are used by the models of this slide type are based only on project information. For more information on project information, see: Project properties dialog
    Chapter title This type of slide is usually used several times in the project to mark the beginning of the chapter.
    The texts variables that are used by the models of this slide type are based on project information and on chapter information.
    Credit title This type of slide is usually used at the end of the project as the end credits.
    The texts variables that are used by the models of this slide type are generally computed. For more information on texts variables, see: Dynamic variables
     
  • Within each category, the predefined title slides are classified into the following types:
    Statics models This type lists statics models ie without animations.
    Animated models This type lists animated models ie with several shots.
    Custom models This type lists the templates that you saved with the Save as model function of the Slide properties dialog box.
     
  • The resources used by the models you create are not moved. So, if you include a photo in a model, the model will only be valid until the picture will not move. This is why it is recommended, to think resources included before creating a new model.
     
  • It will always be possible to create slides from a model using the Add or modify a predefined title slide dialog box then convert it to a standard slide using this function:
    • Convert to standard slide
