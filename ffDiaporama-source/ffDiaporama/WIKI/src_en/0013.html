The big interest of variables is that they are inserted into text (see: Edit text dialog) and are transformed into values ​​only when they are used.
It's why variables are used in many parts of the application (slide, thumbnails, predefined titles slides, video tags, xbmc nfo, etc...)
 
The following table lists all the available variables:
 
Category Variable Content
Project properties values %PTT% Value of the Project title field of the Project properties
%PAU% Value of the Project author field of the Project properties
%PAI% Value of the Project author field of the Project properties displayed as Author presents
%PAP% Value of the Project author field of the Project properties displayed as A Author production
%PAL% Value of the Project album field of the Project properties
%PLN% Value of the Project location name field of the Project properties
%PLA% Value of the Project location address field of the Project properties
%PCT% Value of the Project comment field of the Project properties
%PDD% Duration of the project displayed as hh:mm:ss
%PSC% Project slide count (Number of slide in the project)
%PLD% Long date (The form of the display depends on geographical information provided by the computer, depending on the current language)
%PSD% Short date (The form of the display depends on Short date format selected in the Application options dialog)
%PYR% Year (The year of the event date as a four digit number)
%PMM% Month (The month of the event date as a name, depending on the current language: January, February, etc...)
%PMD% Month as number (The month of the event date as a two digit number)
%PDY% Day (The day of the event date as a two digit number)
%PDW% Day of week (The day of week of the event date as a name, depending on the current language: Monday, Thuesday, etc...)
Current slide values %CSN% Value of the Slide name field of the slide as defined in the Slide settings zone of the Slide properties dialog
%CSR% Current slide number
Chapters values %CCN% Value of the Chapter name field of the latest slide which defined Chapters information. (Chapters information may be defined in Chapter properties dialog or in Predefined title slide dialog).
%CCR% Current chapter number
%CCI% Current chapter number displayed as Chapter N° (depending on the current language)
%CCD% Current chapter duration
%CCT% Chapter count (Number of chapter in the project)
%CLN% Current chapter location name (if one is defined or project's location name).
%CLA% Current chapter location address (if one is defined or project's location address).
%CLD% Long date of the chapter (if one is defined or project's date). (The form of the display depends on geographical information provided by the computer, depending on the current language)
%CSD% Short date of the chapter (if one is defined or project's date). (The form of the display depends on Short date format selected in the Application options dialog)
%CYR% Year of the chapter's date (if one is defined or year of project's date), as a four digit number.
%CMM% Month of the chapter's date (if one is defined or month of project's date) as a name (depending on the current language: January, February, etc...)
%CMD% Month of the chapter's date (if one is defined or month of project's date) as a two digit number.
%CDY% Day of the chapter's date (if one is defined or day of project's date) as a two digit number.
%CDW% Day of week of the chapter's date (if one is defined or day of project's date) as a a name (depending on the current language: Monday, Thuesday, etc...)
Various values %FFD% ffDiaporama version as ffDiaporama 1.7 (20130919)
%CPY% Copyright of the project construct as © Year - Author with Year is the year of today's date and Author is the value of the Project author field of the Project properties
%END% The end (depending on the current language)
%STA% A full text composed of: Project done the short date with «ffDiaporama version» on a Operating system (Number of Core/CPU Core/CPU) computer.
%STP% A full text composed of project statistics (Number of slides, project duration, list of chapters, list of musics, how many photo, vector images, videos and predefined title slides, etc...)
%STM% A full text composed of list of musics
%TLD% Current date as long date format (The form of the display depends on geographical information provided by the computer, depending on the current language)
%TSD% Current date as short date format (The form of the display depends on Short date format selected in the Application options dialog)
%TYR% Current date's year as a four digit number
%TMM% Current date's month as a name (depending on the current language: January, February, etc...)
%TMD% Current date's month as as a two digit number
%TDY% Current date's day as a two digit number
%TDW% Current date's day of week as a a name (depending on the current language: Monday, Thuesday, etc...)
