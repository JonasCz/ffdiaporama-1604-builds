Syntax : ffDiaporama [-lang=] [-loglevel=] [Filename]

Setting Explanation
Filename Filename of the project file to load at application startup
-lang= The language to be used. can take the following values:
  • cz to use Czech
  • de to use German
  • en to use English
  • es to use Spanish
  • el to use Greek
  • fr to use French
  • gl to use Galician
  • it to use Italian
  • nl to use Dutch
  • pt to use Portuguese
  • ru to use Russian
  • zh-tw to use Taiwanese
-loglevel= Adjust the level of messages shown in the console:
  • 1 to show all the messages of Debug, Information, Warning and Error levels
  • 2 to show the messages of Information, Warning and Error levels
  • 3 to show the messages of Warning and Error levels
  • 4 to show only the messages of Error levels
Note: messages appear only if you launched the program from a console (CMD under Windows or Terminal under Linux). It is possible to redirect these messages into a log file by using >.
