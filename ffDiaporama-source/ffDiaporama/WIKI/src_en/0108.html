This dialog allows you to define the information which qualifies the project.
This information will then be registered in the form of TAGs in the rendered video files.

Remarks:

  • The properties of the project are a partial implementation of the MPEG-7 standard on the descriptors of multimedia contents.
  • These properties are written into the video files during the rendering and can be exploited by indexing systems (collection managers) or by Media Center Systems.

Settable fields

Fields Remark
Project geometry This field allows you to select the geometry to be used for the project.

Remark:
  • This field is only editable when the dialog box is used for creating a new project. Subsequently, it is not possible to change this value.
Title The title of the project

Remark:
  • If the id3v2 compatibility option is set in option dialog, this fields is limited to 30 characters
Author The name of the author of the project.

Remarks:
  • This field is often shown by indexing system and Media Center Systems under the term Artist.
  • If the id3v2 compatibility option is set in option dialog, this fields is limited to 30 characters
Album The album of the project

Remark:
  • If the id3v2 compatibility option is set in option dialog, this fields is limited to 30 characters
Language The language of the audio track (ISO 639 standard - 3 characters)
Thumbnail These options allow you to define the thumbnail that will be attached to the video file when rendering
Create a new model from this thumbnail or edit custom thumbnail. Opens the Edit thumbnail dialog.
Customize the thumbnail only for this project (without creating a new thumbnail model). Opens the Edit thumbnail dialog.
Expor thumbnail to a jpeg file. By default, the proposed file name is "folder.jpg". ffDiaporama and many Media Center Systems recognize this file as the icon to be used for the directory that contains it.
Event date The date of the event of the project
Override date Check this box to override the date and offer you to enter the date as a free multi-line text
Event location The location where the project's event took place
  • Click on this button to select a location from the Add or modify a location dialog or using a favorite
  • Click on this button to remove actual location (set to empty)
Comment A free comment which can spread out on several lines.

Remark:
  • Numerous indexing systems or Media Center Systems will read only the 1st line.
Options for speed waves These options sets the value of the velocity curves for use whenever the corresponding options are set to "Project Default"
Transition Defined default speed curve for transitions
Block animations Defined default speed curve for block animations
Image animations Defined default speed curve for image animations

Remarks:

  • Some fields are automatically filled and do not appear in this dialog:
    Timestamp Date and time of generation of the video.
    Composer Contains the version of ffDiaporama with which the video was produced.
    Encoder Contains the version of libavformat (ffmpeg) with which the video was produced.
  • For more compatibility with indexing systems and Media Center Systems, ffDiaporama does not include the whole video TAGs but is closer rather to the standard id3v2 than is the most used by these systems today. Furthermore, the video formats do not support every TAG in the same way.

Chapters

Chapters defined in the project are listed for consultation. (To modify chapters use Chapter button in modify slide dialog.)

The following fields appear:
Fields Remark
# Chapter number
Slide Slide number in wich chapter is defined
Title Chapter title
Start Chapter start position
End Chapter end position
Duration Chapter duration

Remark: Video formats do not support all the chapters in the same way:
Format Remark
3GP/MP4/MKV Full support
AVI/MPG/FLV/OGV/WEBM No support
