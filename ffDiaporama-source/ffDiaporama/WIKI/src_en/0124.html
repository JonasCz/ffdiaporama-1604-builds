This dialog allows you to cut a sound file.

The progress bar

  • Displays as follows:
    • The wave form of the sound is displayed under a white background.
    • The ruler representing the duration of the sound file is dispayed under a black background. Remark than the length of the ruler is increased in proportion to the duration of the sound file.
    • The blue part, on wave form and ruler, represents the part of the sound file which is selected (which will be used in the assembly).
  • This button starts playback. Click again on this button to pause.
  • You can move with the mouse the handle, on the progress bar, to move the current position
  • You can move with the mouse the two and handles, on the progress bar, to move the Start and End positions

The working range

  • Positions the cursor at the starting point
  • Uses the current cursor position as starting point (carries out an edit)
  • Positions the cursor at the end point
  • Uses the current cursor position as end point (carries out an edit)
