This dialog allows you to define the transition duration for a set of slides:
Duration Define the animation duration.

When you click the OK button, the new transition duration is applied to all the slides that were selected.
  • This function can not be used to delete or change the slide's transition.
    If it's what you want, then call the context transitions menu by right clicking on one of the slides of the selection from the timeline and then select "Modify the transition for all the slides of the selection" option.
