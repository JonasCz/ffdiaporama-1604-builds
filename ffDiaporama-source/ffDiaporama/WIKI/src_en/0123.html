This dialog allows you to define a location using Google Maps.

Friendly name Enter a name for this location. This name will be used to display the marker.
Google address Enter the postal adress to find in this zone.
When you press the Enter key, a search is made using Google Maps and the addresses found are used to fill the list.
Then select the correct address in the list
Open the favorite menu:
  • Add to favorite: Adds the current location to the list.
  • Update this favorite: Update the favorite using current settings.
  • Remove this favorites: Remove the favorite from the list
  • The list of the favourite: Use a favorite to fill the dialog box fields.
Friendly address Enter the address in a more readable form. This form will be used to display the marker.
Icon Select an image to be used as icon to display the marker.
Click on the icon zone or on the button to select an image and edit it.
Click on this button to clear the icon.
Map zone Click on a point on the map zone to select it.
You can also use the slider and the and buttons to modify the zoom level.
