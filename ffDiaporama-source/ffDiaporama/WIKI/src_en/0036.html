ffDiaporama uses specific settings for encoding with x264. This settings are including:
- High Quality for "Home Cinema" devices based on x264 main profile, with 3 refs and 3 bframes.
- Phone Quality for mobile devices based on x264 baseline profile, with 3 refs and 0 bframes.
Both settings are using veryfast preset.
Note: Variable bitrate is used with these two codecs. For more information on variable bitrate, see: Render video
