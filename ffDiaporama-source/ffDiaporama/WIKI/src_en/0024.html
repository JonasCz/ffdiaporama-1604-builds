ffDiaporama offers a multimedia file browser which is shown in the main window when you select the file browser mode in the main window. It is designed for quick access to the multimedia files on your computer.
The main advantages are, that you can view the multimedia information of specific files even without or before adding them to your project. For instance, you can filter the view to obtain information of only photos, video files or music files.
And moreover, you can select one or more multimedia files and add them to your project with drag & drop.
The main purpose of the file browser is therefore the viewing and use of (multimedia) files. Some file management tasks are integrated for your convenience, i.e. creating folders deleting files or folders and renaming files or folders.

Remark:

  • Files can be presented differently than by the system file browser (presence of filters showing only some files - management of the file/icon pairs - etc.)
  • The multimedia file browser follows the display rules used by the majority of multimedia systems (gateway, multimedia hard drive, etc.):
    • If a drive or a folder contains a folder.jpg file, then this image file is used as icon for the drive or the folder
    • If a video or music file is accompanied with an image file with the .jpg extension carrying the same name, then this image file is used as icon for this file (example: holidays.mkv and holidays.jpg in the same folder
  • The ffDiaporama multimedia file browser is not designed to manage files: so you cannot make copies of files or move files.

Folder tree

The folder tree shows the drives and folders of your system. Under every drive the folders of that drive are listed, then under every folder the subfolders, if present and so on.
You can:
  • Click a drive or a folder to select it
  • Click the triangle preceding the drive name or folder name to display or hide the list of subfolders
  • Make a right click on a drive or a folder to call the contextual menu on this drive or folder. The contextual menu proposes then the following options:
    • Refresh all to update the whole tree (as so if you have to insert a CD-ROM, a DVD or an USB drive, this one is shown in the tree)
    • Refresh from here to update the tree from this folder (update only what is under this folder)
    • Create a new folder
    • Remove a folder
    • Rename a folder

The current folder view lists files and subfolders present in the current folder.
Files are shown:
  • Either as a table with numerous information if you selected the details view in the settings menu of the browser
  • Or as image if you selected the icons view in the settings menu of the browser

Files are shown according to the filter of file selection in the settings menu of the browser
You can:
  • Select a file or a folder by clicking it
  • Select several files or folders by using SHIFT and CTRL keys
    • CTRL+Click on a file: Add to the current selection or remove from the current selection
    • SHIFT+Click on a file: Select all the files between the last selected and the one which you have just clicked
  • Double click a file or a folder:
    • If it's a folder: select it as the current folder and opens it
    • If it's a file: open the program that is by default associated with the file type in the system. For example, if you have associated GIMP with .jpg files in the system file browser, then a double click on a .jpg file will open the file with GIMP. (Remark: A double click on a ffDiaporama project file, opens the ffDiaporama project)
  • Perform a right click on one of the files or folders of the selection to open the contextual menu. According to the elements that are present in the selection, the menu will propose different actions

You can also use Drag & Drop to move selected files towards the timeline to:
  • Add them to the project, if they are images, videos or ffDiaporama project files
  • Use them as playlist, if they are audio files

Status bar to the current selection

The status bar to the current selection shows information about one or several selected files.

For a single file selection, the following information is shown:
Image of the file Filename (Filesize)
According to the file type:
Image: Information about the size of the image / copyright / camera / orientation
Video: Information about the format of the images / the video track / the audio track / duration
Music: Information about the audio format / duration
ffDiaporama project: Version of ffDiaporama / Format / Number of slide and chapter / duration of the project
According to the file type:
Image : Information about the picture shooting (Aperture and Focal / Lens / With or without flash)
Video: TAG contents (Title / Artist / Album / Year)
Music: TAG contents (Title / Artist / Album / Year)
ffDiaporama project: TAG contents (Title / Artist / Album / Year)

For a multiple files selection, the following information is shown:
Icon of the file type Number of files and type of files
Durations accumulated by files
Size accumulated by files

Status bar of the current folder

The status bar of the current folder shows the following information about the current folder:
Number of files Number of sub-folders
Total size of shown files Total duration of shown files
Space occupied on the disk / Total space of the disk

Favorites menu

The list of the favourite folders can be managed in the favorites menu.
  • First line of the menu: Add to favorite adds the current folder to the list. Then an edit box allows you to give a name for this favorite
  • Between the first and the last line: The list of favorites folders is displayed. Select one of the favorite folders to access it directly
  • Last line of the menu: Manage favorite allows to rename or to delete entries of the list. A dialogbox for the modification of the list is displayed

Settings menu

The settings menu allows to modify the display of the current folder:

Settings of the display mode
Details view Switch the display of the current folder to detail mode. In this mode, the present elements in the folder are shown in a table with numerous information
Icon view Switch the display of the current folder to icon mode. In this mode, the present elements in the folder are shown as images
     

Settings of the sort order
Sort by number This option sorts the files in order of their numerical suffix : Example: if you select IMG_0025, IMG_0032, MVI_0029 and MVI_0040, the files will be inserted as follows : IMG_0025, MVI_0029, IMG_0032 and MVI_0040
Sort by date This option sorts the files in order of their creation date
Sort by name This option sorts the files in alphabetical order
Sort by date This option sorts the files in order of last modification date
Show folder first If this option is enabled, the directories will be displayed first
     

Settings of the filter on files
All files Show all the present elements in the current folder
Managed files Limit the display to the usable elements of the current folder:
  • Image, video and audio files of which the format is recognized by ffDiaporama
  • Subfolders
Image files Limit the display of the current folder to the image files of which the format is recognized by ffDiaporama in the present subfolders
Image vector files Limit the display of the current folder to the image vector files of which the format is recognized by ffDiaporama in the present subfolders
Video files Limit the display of the current folder to the video files of which the format is recognized by ffDiaporama in the present subfolders
Audio files Limit the display of the current folder to the audio files of which the format is recognized by ffDiaporama inthe present subfolders
ffDiaporama projects Limit the display of the current folder to the ffDiaporama project files in the present subfolders
     

Additional settings
Display hidden files and folders Show or hide the hidden files and folders.
Under Windows, hidden files and folders have the Hide file attribute activated.
Under Linux, hidden files and folders have a name beginning with a dot "."
This option accumulates both versions for both systems. So, even under Windows, files and folders with a name beginning with a "." are considered as hidden files or folders.
Remark: or One of this icon is displayed in front of the name of the option as this one is activated or not.
Hide filename This option is available only for the icon view mode. If this option is activated, the file names will not be shown under the file images. It allows to increase the number of images shown in the view of the current directory.
Remark: or One of this icon is displayed in front of the name of the option as this one is activated or not.
