Para que funcione, o programa necesita dous ficheiros de configuración e unha base de datos. Os dous ficheiros de configuración compóñense dun par de ficheiros.

Por exemplo:
  • O ficheiro ffDiaporama.xml principal almacenase no directorio do ffDiaporama. Este ficheiro é global para todos os usuarios: contén os parámetros predeterminados do programa ffDiaporama.
    • Para Windows, é C:\Program Files\ffDiaporama
    • Para Linux, é /usr/share/ffDiaporama

  • Un ficheiro secundario ffDiaporama.xml específico do usuario está situado no directorio de perfil de usuario.
    • Para Windows, é C:\Document and settings\[usuario]
    • Para Linux, é ~/.ffDiaporama

Cando se inicia o programa por primeira vez, créanse os ficheiros de configuración do usuario e da base de datos se aínda non están dispoñíbeis.

Observación:

Se ffDiaporama é utilizado por varios usuarios no mesmo computador, haberá ficheiros de configuración e de base de datos independentes para cada usuario.

Ficheiro Uso
ffDiaporama.xml Contén as opcións do programa para ffDiaporama
Devices.xml Contén a base dos perfiles para a xeración de vídeo.
ffDiaporama.db É unha base de datos SQLite3 que contén obxectos na caché e algunhas opcións de configuración.
