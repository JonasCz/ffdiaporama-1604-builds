Este diálogo permítelle modificar o texto dentro dun bloque.

Propiedades para todas as tomas

As modificacións realizadas sobre as seguintes propiedades aplicaranse a todas as tomas da diapositiva.
Para entender realmente as diferenzas entre as propiedades que se aplican a todas as tomas e as que se aplican só á toma actual, consulte: Como traballa o ffDiaporama.

Lapela de texto

Primeira liña na parte superior do diálogo

Este grupo de axustes é o obxecto da xestión do estilo. Para obter máis información sobre a xestión do estilo, consulte: A xestión de estilos
Permite seleccionar o tipo de letra a empregar. Este axuste aplícase á selección actual.
Permite estabelecer (ou retirar) unha lista de viñetas. Este axuste aplícase aos parágrafos da selección actual.
Permite estabelecer (ou retirar) unha lista numerada. Este axuste aplícase aos parágrafos da selección actual.
Aliñamento horizontal: á esquerda. Este axuste aplícase ao paragrafo no que está o cursor.
Aliñamento horizontal: centrado. Este axuste aplícase ao paragrafo no que está o cursor.
Aliñamento horizontal: xustificado. Este axuste aplícase ao paragrafo no que está o cursor.

Segunda liña na parte superior do diálogo

Permítelle engadir contorno ou sombra a todo o texto:
  • A primeira lista permítelle seleccionar o efecto:
    Sen efectos O texto amósase sen ningún tipo de sombra ou contorno
    Contorno Rodea os caracteres cunha liña na cor actual de sombra
    Sombra ... Engadir unha sombra na cor actual de sombra nun dos 4 cantos.

  • A segunda lista permítelle seleccionar a cor da sombra ou o contorno.
Estabelece os caracteres en negra. Este axuste aplícase á selección actual.
Estabelece os caracteres en itálica. Este axuste aplícase á selección actual.
Subliña os caracteres. Este axuste aplícase á selección actual.
Pon os caracteres como superíndice. Este axuste aplícase á selección actual.
Pon os caracteres como subíndice. Este axuste aplícase á selección actual.
Move o parágrafo un nivel á dereita (sangría). Este axuste aplícase ao paragrafo no que está o cursor.
Move o parágrafo un nivel á esquerda (elimina a sangría). Este axuste aplícase ao paragrafo no que está o cursor.
Aliñamento vertical: arriba. Este axuste aplícase ao bloque (para todo o texto).
Aliñamento vertical: medio. Este axuste aplícase ao bloque (para todo o texto).
Aliñamento vertical: abaixo. Este axuste aplícase ao bloque (para todo o texto).
Permítelle engadir unha variábel dinámica ao texto. Para obter máis información sobre as variábeis dinámicas vexa: Variábeis dinámicas

Lapela do fondo do texto

Nota: Esta lapela só está dispoñíbel para os bloques de título (Non está dispoñíbel cando o texto é aplicado a un bloque de imaxe).

Este grupo de axustes é o obxecto da xestión do estilo. Para obter máis información sobre a xestión do estilo, consulte: A xestión de estilos
Permite definir un deseño de fondo para a forma. Están dispoñíbeis as seguintes opcións:
Sen deseño A forma non terá fondo polo tanto será totalmente transparente.
Deseño sólido Permite seleccionar unha cor uniforme.
Deseño de patrón Permite seleccionar un deseño raiado.
Degradado de 2 cores Permite seleccionar un degradado de 2 cores.
Degradado de 3 cores Permite seleccionar un degradado de 3 cores.
Imaxe desde a galería Permite seleccionar unha imaxe da galería.
Se o deseño do fondo é de calquera dos tipos, sólido, patrón o degradado, entón aparece esta liña e permite seleccionar unha cor. (Dependendo do fondo que seleccione, haberá unha, dúas ou tres cores seleccionábeis.)
Se o deseño do fondo é do tipo patrón, entón aparecerá esta liña para permitirlle seleccionar un patrón.
Se o deseño do fondo é un degradado, entón aparecerá esta liña para permitirlle seleccionar a orientación do degradado.
Se o degradado ten 3 cores, controles adicionais permiten definir a posición (en porcentaxe) da 3Âª cor do degradado.
Se o deseño do fondo é unha imaxe da galería, entón aparecerá esta liña para permitirlle seleccionar unha imaxe.

Lapela de marxes

Marxes predeterminados Para utilizar as marxes predefinidas para a forma do bloque. En xeral, estas marxes aseguran que o texto está contido na forma.
Forma completa Para utilizar marxes correspondentes á forma completa, de feito, a forma de rectángulo seleccionada é independente do deseño.
Personalizado Para utilizar marxes personalizados. As seguintes opcións permiten axustar as marxes:
Representa a posición horizontal das marxes relativas ao canto superior esquerdo do bloque. (Este valor expresase no % do largo do bloque.)
Representa a posición vertical das marxes relativas ao canto superior esquerdo do bloque. (Este valor expresase no % da altura do bloque.)
Representa o largo das marxes. (Este valor expresase no % do largo do bloque.)
Representa o alto das marxes. (Este valor expresase no % do alto do bloque.)
