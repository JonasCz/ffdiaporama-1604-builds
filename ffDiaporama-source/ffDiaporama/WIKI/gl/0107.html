Esta xanela de diálogo permítelle exportar o proxecto e todos os recursos utilizados polo proxecto a un cartafol
 
Cartafol de destino Permítelle seleccionar o cartafol base de destino. Prema no botón do cartafol para abrir o selector de directorio (cartafol).
Subcartafol do proxecto Permítelle seleccionar o nome do cartafol que se creará para os ficheiros do proxecto. De xeito predeterminado proponse un título para o proxecto.
 
Unha vez completada a selección, prema no botón Aceptar para iniciar la exportación.
En calquera momento durante este proceso, pode premer no botón Cancelar para deter a exportación.

Observacións:

  • A exportación ten en conta todos os recursos que compoñen o proxecto:
    • Imaxes
    • Vídeos
    • Música
    • Elementos miniatura
    • Fondos de pantalla
    • imaxe predeseñada que non se almacenan baixo o alias de ClipArt.

  • As exportacións non só copian os ficheiros. Aplícanse as seguintes regras:
    • Se hai varios ficheiros de diferentes directorios que teñen os mesmos nomes, estes ficheiros son renomeados.
    • Crease un ficheiros de proxecto .ffd tendo en conta os cambios no directorio e o nome do ficheiro.
