Este diálogo permite comprobar a configuración de ferramentas externas e bibliotecas utilizadas por ffDiaporama.
Isto é especialmente útil para os usuarios de Linux e para aqueles que compilan o programa.
Os erros máis comúns dos que se amosan refírense aos formatos admitidos, códecs e versións de bibliotecas.

Para resolver algúns problemas haberá que instalar paquetes adicionais no computador e compilar ffDiaporama desde o código fonte, e comprobar de novo a configuración.
