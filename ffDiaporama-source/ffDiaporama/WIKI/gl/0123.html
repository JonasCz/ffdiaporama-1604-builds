Este diálogo permítelle definir unha localización empregando Google Maps.

Nome: Escribir un nome para esta localización. Este nome utilizase para amosar o marcador.
Enderezo de Google Introducir o enderezo postal para buscar nesta zona.
Cando preme na tecla Intro, realizase unha busca a través de Google Maps e os enderezos atopados utilízanse para encher a lista.
A seguir, seleccione o enderezo correcto na lista
Abrir o menú de favoritos:
  • Engadir a favoritos: engade a localización actual á lista.
  • Actualización deste favorito: Actualizar o favorito usando a configuración actual.
  • Retirar este favorito: Retirar o favorito da lista
  • A lista de favoritos: Utilizar un dos favoritos para encher os campos da xanela de diálogo.
Enderezo amigábel Introducir o enderezo nunha forma máis lexíbel. Este formulario utilizase para amosar o marcador.
Icona Seleccionar unha imaxe que se utilizará como icona para amosar o marcador.
Prema na icona da zona ou no botón para seleccionar unha imaxe e editala.
Prema neste botón para borrar a icona.
Zona do mapa Prema nun punto da zona do mapa para seleccionalo.
Tamén pode usar o control esvárante e os botóns para modificar o nivel de zoom.
