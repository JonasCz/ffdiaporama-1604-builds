Syntaxe : ffDiaporama [-lang=] [-loglevel=] [NomeDeFicheiro]

Axuste Explicación
Nome de ficheiro Nome do ficheiro de proxecto para cargar ao inicio do programa
-lang= O idioma que usar pode ter os seguintes valores:
  • cz para o checo
  • de para o alemán
  • en para o inglés
  • es para o español
  • el para o grego
  • fr para o francés
  • gl para o galego
  • it para o italiano
  • nl para o holandés
  • pt para o portugués
  • ru para o ruso
  • zh-tw para o taiwanés
-loglevel= Axustar o nivel das mensaxes que se amosan na consola:
  • 1 para amosar todas as mensaxes de depuración, información, aviso e os niveis de erro
  • 2 para amosar as mensaxes de información, aviso e os niveis de erro
  • 3 para amosar as mensaxes de aviso e os niveis de erro
  • 4 para amosar só os niveis de erro
Nota: As mensaxes aparecen só se o programo foi iniciado desde unha consola (CMD en Windows ou Terminal en Linux). É posíbel redirixir estas mensaxes a un ficheiro de rexistro usando o carácter «>».
