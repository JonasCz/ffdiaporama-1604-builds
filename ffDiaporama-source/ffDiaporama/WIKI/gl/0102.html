Esta xanela de diálogo permítelle modificar a base de datos de perfiles de equipos.
 
A base de datos componse realmente de dúas partes:
A base de datos predeterminada (fornecida co ffDiaporama). Estes dispositivos están rexistrados no ficheiro de configuración global de ffDiaporama.
Os dispositivos que modifique ou engada. Estes atópanse rexistrados no ficheiro de configuración persoal.

A zona de tipo de dispositivo permítelle seleccionar o tipo de equipo que é amosado na lista.

A lista amosa as categorías e os tipos de dispositivos presentes na base de datos.
Os dispositivos precedidos por esta icona están definidos na base de datos predeterminada.
Os dispositivos precedidos por esta icona son os definidos por vostede, ou os da base de datos predeterminada que foron modificados.

Os seguintes botóns permiten actuar sobre a base de datos:
Engadir dispositivo: Permítelle crear un novo dispositivo para engadilo na base de datos.
A creación dun novo dispositivo faise en 3 pasos:
  1. Primeiro ten que crear o dispositivo
  2. Definir varios parámetros
  3. Premer o botón «Aplicar as modificacións» para rexistrar os parámetros
Retirar o dispositivo: Permítelle eliminar o dispositivo.
Teña en conta que só pode eliminar os dispositivos que teña creado, que son os que se amosan con esta icona:
Restabelecer a valores predeterminados: Cancela as modificacións que se fixeron nun dispositivo definido na base de datos predeterminada no ffDiaporama
 

Observacións:

Subtipo de dispositivo Permítelle seleccionar a categoría na que clasificar o equipamento
Modelo do dispositivo Introduza aquí o nome do dispositivo (campo de entrada de texto libre) e, a seguir, introduza todos os parámetros necesarios para a xeración de vídeos que estean asociados co perfil do dispositivo (formato do ficheiro, norma de vídeo, etc)
